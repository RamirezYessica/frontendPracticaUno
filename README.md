# Lista de tareas

## Descripción 
Esta aplicación permite:
- Crear tareas
- Listar tareas
- Marcar tareas como completadas (se eliminan de la lista)

Se comunica con una API REST desarrollada en .NET mediante **HttpClient**.

## Requisitos para ejecutar
- Node.js (v18 o superior)
- Angular CLI

1. Ubicarse en la carpeta de frontend
    cd frontend
2. Instalar dependencias
    npm install
3. Levantar la aplicación 
    ng server
